23#21 New Objective, Work Harder, Be More Efficient

Posted on 2023-05-27 Edited on 2023-06-25 In WeeklyReport Word count in article: 2.8k Reading time ≈ 3 mins.

慎终如始，则无败事。 (《老子 · 德经》)

To improve efficiency, it is time to set a goal. September 9th is the target date. In addition, I will try my best to catch up with AAAI2024. There are only 81 days left, so I'll need to work hard.

Here's a summary of last week's progress.

  • Habits:

  • Morning:

  • Wake up at 6:30 am every day

  • Daily English Words

  • Daily Abstract: learn how top researchers write the abstract

  • Daily Paper reading (read at least 1 paper every day carefully and record it in Notes)

  • Noon:

  • Daily Oral: Practice oral skills every day, especially technical terms. Address this weakness promptly!
    Evening:

  • Daily Leetcode practice and weekly contest

  • Daily exercise (swim at least twice, get up early and jog, and keep fit)

  • Daily Coursera: on the one hand, review DL basics; on the other hand, complete an important task every day.

  • Daily New Knowledge: whatever kinds of new knowledge, it can be math, computer, economics, history, and music. Keep eager, Keep curious.

  • Life:

  • Make sure I get sufficient sleep (sleep before 11 pm and have a noon break)

  • Learning:

  • Review English grammar (Book: 101 essays, 英语语法新思维)

  • Review Python and Pytorch

  • Review fundamental machine learning and deep learning algorithm (In English and Pay Attention to the terms!!)

  • Research:

  • Update wolai personal page and use English to record everything

  • Spend enough time

  • Conduct experiments on base models

  • Read more literature and think about how to do backdoor attack

  • Application:

  • Update personal website

  • Prepare the oral presentation

This week, I wake up at 6:30 am daily and take a one-hour break at noon. I make sure to get enough sleep to handle my research, but I realize that I need to plan my tasks better. Unfortunately, I didn't exercise enough this week because I spent most of my time researching and studying.
