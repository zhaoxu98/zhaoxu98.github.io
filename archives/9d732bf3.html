23#19 Keep Going and Manage the Time

Posted on 2023-05-13 In WeeklyReport Word count in article: 2.3k Reading time ≈ 2 mins.

I came to HKUST GuangZhou campus this Tuesday. This is a beautiful university and the group members are very kind. I like conducting research here! Although it is very busy this week, I learned a lot and enjoyed everything. This is a good state for me.

However, one of the mistakes I made is that I did not balance my time and did not achieve my goal as planned. Besides, I need to adjust my current timetable to make sure I can finish everyday self-study tasks as well in this new environment.

我认为保持每天在习惯培养和自学新知识上的时间投入是有必要的，尽管本月的科研任务和申请事务相当繁重。同时，充足的睡眠也是很重要的，这能够保证我的学习效率。因此，可以适当缩减在娱乐方面的投入并尽可能地再提高效率。

在时间分配方面，首先，我觉得要保持 23 点入睡与 6 点半前起床。同时，经过本周的实践，我发现在午饭后的一个小时内困意的确会对学习效率有较大影响，所以可以将这段时间用来回宿舍午休（在实验室无法平躺减轻对腰椎压力），尽管这会带来额外的来回时间开销。因此，理论上我可以保证 8 个小时的睡眠。

其次，目前比较紧急的任务是申请事务和科研课题。申请方面主要有比较多的文书要写，而科研方面则需要大量的文献阅读、总结和代码学习。我认为必须要对得起刘老师的信任，所以必须平等地保证在这两方面的时间投入，这也就意味着我需要严格制定计划并严格执行。

此外，每天的习惯养成方面：

  1. leetcode 每日一题还在坚持，每天早上就完成了，但是从质量方面来看，对新知识的学习有所懈怠。特别是图论和树的算法题方面，如周赛 Hard 级别的难题，没有将算法完全搞懂。
  2. 这周因为要熟悉新环境，所以没有坚持每天锻炼，下周开始必须要保证，从每天晚上去跑步开始做起吧，也可以将健身环也做起来。
  3. 每日英语学习完全没有按照计划执行，主要表现为没有按计划背单词和做口语练习，而且临考试才磨枪，所以在考试时发现了不少问题。下周开始，每天保证 81 个新单词的学习，也要及时复习旧单词和阅读文献时积累的科研词汇。同时，将 abstract 的背诵也加入计划，用这样的笨办法尽快上手科研表达。
  4. 每日 Coursera 在上周完成地不错，目前已经完成了 Deep Learning Specialization 的第 2、3 门课程的学习并获得了证书。但从周二开始一直没有机会完成，接下来要坚持。
  5. 坚持每天一篇文献的阅读、英文总结。

具体地，我认为可以这样安排时间：

Time TODO
7:30 洗漱、早餐和完成每日一题
7:30 - 8:00 学习新单词、复习科研词汇和 abstract 背诵
8:00 - 10:00 Coursera 学习、新知识学习 (like Coding and Grammar) 、文献阅读
10:00 开始当日任务 (按前一天的计划表)
After Lunch 午休一小时以内
21:00 - 21:40 根据当天情况自行选择回去时间，先去锻炼身体
Before 23:00 制定计划、休息
